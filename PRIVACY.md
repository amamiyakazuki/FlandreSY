# 隐私说明

芙兰水衣没有自建后端，也不会把你的账号信息上传到作者的服务器。

## App 会保存什么

App 可能在本机保存这些数据：

- 住理生活手机号和设备码缓存
- U净登录状态和 token
- 本地洗衣机快捷入口
- 饮水历史记录
- 洗衣订单本地历史记录
- 本地运行日志

这些数据用于让 App 记住登录状态、恢复当前订单、展示设备列表和统计历史。

## App 会请求哪些权限

不同 Android 版本上权限名称可能略有不同，主要包括：

- 相机：扫码。
- 蓝牙：热水设备控制。
- 定位：Android 蓝牙扫描通常需要定位权限配合。
- 通知：显示运行状态或后台操作提示。
- 网络：请求相关服务接口和检查 GitHub Release 版本。

## 数据会发到哪里

App 会请求住理生活、U净及 GitHub Releases 等相关接口。除此之外，项目没有自建统计服务，也没有埋点分析服务。

## 你可以怎么清理数据

如果你想清理本地数据，可以：

- 在系统设置里清除 App 数据。
- 在 App 内删除本地设备快捷入口。
- 卸载 App。

## 重要提醒

不要把自己的日志、抓包文件、token、订单信息随便发到公开仓库或群聊里。出了问题先脱敏，再讨论。
