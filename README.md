# 一键热水工具

这是住理生活热水设备的最小验证版 Android 工程。

## 怎么运行

1. 用 Android Studio 打开这个目录：`ZhuliHotwaterTool`
2. 等 Gradle 同步完成
3. 连接 Android 手机，开启蓝牙和定位
4. 运行到手机
5. 输入住理生活账号密码，设备号默认是 `1006445`
6. 站在设备附近，点“登录并开水”

## 第一次测试看什么

界面日志会显示每一步：

- 登录成功
- 获取设备信息
- 扫描 BLE 设备
- GATT 已连接
- 握手完成
- 订单已创建
- 启动确认完成

如果失败，把最后 5 到 10 行日志发回来，不要发密码。

## 当前实现边界

- 只支持账号密码登录
- 只按本人账号、指定设备正常扣费流程调用
- 热水设备按 APP 逻辑传 `money=0` 和 `consume_value=0`
- 已加“结束热水”按钮，避免只开不停
