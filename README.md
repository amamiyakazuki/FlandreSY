# 芙兰水衣

一款面向住理生活场景的 Android 工具 App，主要接入：

- 住理热水
- U净洗衣
- U净饮水

它的目标很直接：把真实业务流程做稳，把界面做清楚，让每一步都能看得明白一点，少一点糊里糊涂的小坑。

## 运行方式

1. 用 Android Studio 打开本仓库
2. 等 Gradle 同步完成
3. 连接手机或模拟器，运行 `app`
4. 按页面提示登录、扫码、创建订单、支付或启动流程

## 主要能力

- 住理热水登录、开热水、关热水
- U净洗衣验证码、登录、扫码、下单、支付宝支付、启动/停止、订单刷新
- U净饮水扫码、确认设备、创建接水订单、刷新订单
- 首页、订单页、设备页联动当前运行状态

## 说明

- 当前用户侧只把支付宝作为洗衣支付主路径
- 支付成功不等于洗衣机已启动，订单详情仍会继续刷新
- 真实账号、token、session 不会写进普通日志
- `P_PLAN/`、`screenshots/`、`.codex/` 之类本地过程文件已忽略

## 备注

如果某一步失败，优先看页面状态和日志，不要急着重试太多次，先把失败原因抓清楚会更省力一点。
